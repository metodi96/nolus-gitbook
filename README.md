# Nolus Documentation

Welcome to the Nolus documentation hub. The content here is organized into five sections:

- **Protocol Documentation** — concepts, mechanics, and risk model of the Nolus protocol (sourced from the [Nolus Knowledge Hub](https://hub.nolus.io/en/collections/10034429-tech-documentation)).
- **Networks & Operators** — guides for running a full node, validator, or relayer on Pirin (mainnet) and Rila (testnet).
- **JavaScript SDK** — how to use [`@nolus/nolusjs`](https://github.com/nolus-protocol/nolus.js) to interact with the protocol from JavaScript or TypeScript.
- **Solray SDK** — coming soon.
- **Agentic Workflows** — using the Nolus MCP server to let AI agents interact with the protocol.

Use the sidebar to navigate. The narrative content is curated; the auto-generated `@nolus/nolusjs` API reference lives upstream at [nolus-protocol.github.io/nolus.js](https://nolus-protocol.github.io/nolus.js/) and is linked from the SDK pages.
